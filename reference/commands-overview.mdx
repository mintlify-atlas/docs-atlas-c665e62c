
## Global flags

These flags work with all commands.

| Flag | Type | Description |
|------|------|-------------|
| `--config <path>` | string | Path to config file (default: auto-detect) |
| `--no-config` | boolean | Run without any config file |
| `--version` | boolean | Print VectorLint version |
| `--help` | boolean | Print help for the command |

## Getting help

```bash
vectorlint --help
vectorlint lint --help
vectorlint init --help
vectorlint rules --help
```
